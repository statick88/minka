# Minka Project

## Quick Start

1. **Configurar**:
   ```bash
   ./scripts/setup.sh
   ```

2. **Editar variables de entorno**:
   ```bash
   vim docker/.env
   # Añade tu GITHUB_TOKEN
   ```

3. **Iniciar**:
   ```bash
   ./scripts/start.sh
   ```

4. **Usar**:
   ```bash
   docker-compose -f docker/docker-compose.yml exec minka bash
   minka start
   ```
